## Como compilar
Para os códigos das pilhas, faça

	g++ -Wall -std=c++11 -I include/ src/stackMain.cpp src/stackint.cpp -o bin/stack

Para o código da fila, faça

	g++ -Wall -std=c++11 -I include/ src/queueMain.cpp -o bin/queue
